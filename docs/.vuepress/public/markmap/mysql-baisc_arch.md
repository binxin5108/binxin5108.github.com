# **Mysql**

### **连接器**
- 建立连接、获取权限、维持和管理连接
- 长连接、段连接
### 查询缓存
- 往往弊大于利，8.0已移除
### **分析器**（做什么）
#### 词法分析
- SQL 里每个字符串分别是什么，代表什么
#### 语法分析
- 根据语法规则，判断输入的 SQL 是否满足 MYSQL 语法
### **优化器**（怎么做）
- 决定使用哪个索引、决定表的连接顺序
### **执行器**
- 操作引擎，返回结果
### **存储引擎**
- 负责数据存储和提取
- 插件式：InnoDB、MyISAM、Memory
- 不同引擎共用一个Server层
